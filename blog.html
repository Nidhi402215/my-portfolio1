June 28, 2025

Creating my first website was an exciting and valuable learning experience. I started by planning the structure of the website, deciding that I wanted it to be a personal portfolio to showcase my skills, projects, and background. Once I had a basic idea of what I wanted, I began writing the HTML code to build the foundation. This included creating different sections like the homepage, about me, projects, and contact information. I used simple HTML tags to add headings, paragraphs, images, and links.

After building the structure, I moved on to styling the website using CSS. I chose a clean and modern design, picking suitable fonts, colors, and layouts. I experimented with different styles and made use of CSS features like flexbox and grid to align elements properly and create a responsive layout that would look good on both desktops and mobile devices.

To add some interactivity, I used JavaScript. For example, I included some simple animations to make the website feel more dynamic. I also learned how to center images, align text, and make my website visually appealing.

Throughout the process, I used Visual Studio Code as my code editor and tested the website using the Live Server extension, which helped me see the changes instantly. It was a journey of learning through trial and error, but it gave me a solid understanding of how websites are built from scratch. Seeing my website live on the browser for the first time was truly rewarding, and it motivated me to keep learning and improving my web development skills.
